Tree-Drawer Class Demo Site

This site uses the CSSTree.js class to draw a configurable, stylable, visual representation of a tree-based set of data. Source code for the class and a basic jquery wrapping plugin can be found in the scripts/src folder of the CSSTreeDrawer project.

Use this site to test out different settings options. Look at the individual user, default, and compiled settings that get compiled and displayed at the bottom to check out how the settings are structured. Copy the unformatted user settings to pass as an argument for your own usage of a CSSTree.

Node HTML Template

Choose the HTML template to be used for each node. Select 'Default' or leave blank / delete all text to use the default settings. Select 'User' and define your own HTML template function for customization, or choose one of the example functions to pair with its corresponding example data.
